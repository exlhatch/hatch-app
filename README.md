# HATCH — Guiding Intelligence

Live chalkstream hatch forecasting app.

## Live Data Sources
- **Environment Agency Hydrology API** — real-time river level, flow, temperature
- **Open-Meteo** — 7-day hourly weather forecasts

Both APIs are free and require no API keys.

## Deploy to Vercel

1. Push this folder to a GitHub repository
2. Go to vercel.com and sign in with GitHub
3. Click "Add New Project" and select your repo
4. Click "Deploy" — no settings changes needed
5. Live in ~60 seconds

## Local Development

```bash
npm install
npm run dev
```

Opens at http://localhost:5173
